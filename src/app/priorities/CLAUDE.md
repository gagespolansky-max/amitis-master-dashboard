# Priority Board — CLAUDE.md

AI-ranked Kanban board with three columns: This Week / This Month / On Deck. Supports drag-drop reordering (via @hello-pangea/dnd) and pinning.

## Owns

- **_components/:** priority-board.tsx
- **api/:** reads/writes `data/priorities.json`

## Status

Active.

## Connections

- Priorities generated by `python scripts/refresh-priorities.py` which pulls from Gmail + Attio, ranks via Claude, writes to `data/priorities.json`
- API route auto-saves board state back to the same JSON file
- No Supabase dependency — file-based persistence only
